Rock Paper Scissors

This page has been set up so that two players can play the common game of Rock Paper Scissors (also known as ro-sham-bo). The rules are straightforward, but in case you don’t know them:

Rock Paper Scissors is a game involving two players making pre-defined hand gestures at each other. The gesture that each player uses is played against the other, with a winner being decided based on the rules being used.

The three gestures used in base Rock Paper Scissors are... well... rock, paper, and scissors. The way these are scored is as such:

  • Rock beats Scissors
  • Scissors beats Paper
  • Paper beats Rock
Rock Paper Scissors Kata

In reality, the two players would count to three and reveal their move. This page already has buttons to indicate which move the players made. We would like you to complete the game by writing some code to work out who the winner is.

What you need to do

There are two files in this project—this ‘index.html’ file which contains the structure of the page, and a file named ‘roshambo.js’ which contains the code that runs the game. You will need to change the JavaScript file to make sure that once both moves have been entered, the correct winner is displayed in the page. Remember to refresh the page after changing the code, otherwise your changes will have no effect. If you have problems with your JavaScript, you might like to use the developer tools in your browser. You can access Chrome’s Dev Tools by pressing `Command+Option+J` (Mac) or `Control+Shift+J` (Windows etc).

Once you have the game correctly determining the winner, or a draw, then you could look at improving your code. You can also extend your game to include two new moves: lizard and spock. You can find the rules on Rock Paper Scissors Spock Lizard.
